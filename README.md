# EUDR Coffee Traceability & Compliance Platform  

## Project Overview
A full-stack data intelligence system designed to automate EU Deforestation Regulation (EUDR) compliance for 2,000+ coffee plots.  

This platform transforms operations from manual, error-prone spreadsheets into a professional digital ledger, ensuring 100% audit readiness for European export markets.

---

## The Problem: Export Barriers

### Shipment Seizure Risk  
Mixed lots from non-compliant farms can cause entire containers to be rejected at EU ports (Hamburg, Antwerp).

### Data Fragmentation  
Managing thousands of GPS coordinates in Excel leads to:
- Data leakage  
- Inconsistent records  
- Rejected Shipping Instructions (SI)

---

## Real-World Solutions

### Digital Traceability  
- Migrated 2,000+ records into a structured MySQL database 
- Established a Single Source of Truth for export compliance  

### Geospatial Risk Mapping  
- Identified high-risk areas:
  - Aroresa
  - Bensa
- Detected 5.6% failure rate ("Shipment Blockers") 
- Enabled pre-booking lot segregation

### Premium Plot Auditing  
- Prioritized certification for:
  - High-altitude coffee (2000m+)  
  - Regions: Gedeb, Yirgacheffe  
- Protected high-margin specialty coffee contracts

### Executive Command Center  
- Built a treamlit dashboard providing:
  - Real-time monitoring  
  - Export-ready hectare tracking  
  - Decision support for senior management

## Strategic Recommendations

### Lot Segregation  
Use the Woreda Risk Table to:
- Isolate high-risk coffee  
- Prevent contamination of compliant export batches  

### Digital B/L Integration  
- Link compliance certificates directly to Bill of Lading (B/L)  
- Improve customs clearance efficiency with major carriers (MSC, Maersk)

### Field Automation  
- Replace paper-based workflows with real-time database syncing  
- Ensure 100% accurate daily field reports
